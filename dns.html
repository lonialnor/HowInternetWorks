DNS [Domain Name System]

The DNS stands for Domain Name System; it is best described as “the phonebook of the Internet”. Web browsers use IP Addresses and people use domain names, and the DNS manages them both. Every device has a unique IP address that other machines can use to find it. The DNS makes it so people do not have to memorize the long complicated numbers of an IP Address.

It works by converting a host name into an IP address. So when a user puts in, for example, Google.com the DNS will translate it into an address so the computer knows where to go.

There are four DNS servers that load a webpage. There’s the DNS recursor, the root nameserver, the TLD nameserver, and the Authoritative nameserver. The recursor is a server made to receive queries from machines through applications like web browsers. The root server is the first step in translating. The TLD (which stands for “top level domain server” is the next step. It hosts the extension of the domain name; for example, “.com”. Lastly, the Authoritative name server returns the IP address to the recursor.
